# Mecart Blog Article Mockup — Industrial Noise Control

Interactive HTML mockup for the Mecart website redesign blog article template.

**Open `index.html` in a browser to preview.**

## Features
- Reading progress bar
- Sticky table of contents sidebar
- Chapter marker (sticky section indicator)
- Expandable/collapsible H2 sections
- Tabbed content (Regulations section)
- Hierarchy accordion (Controls section)
- Animated bar chart (Noise by Industry)
- Stat counter animations
- FAQ accordion
- Two-column layout
- Contrast panel (Without/With)
- Dual CTA footer
- Pull quote
- Highlighted text annotations
- Drop caps
- Micro-icons on H3s
- Fully responsive (test by resizing browser)

## For the development team
This is a visual and behavioral reference. All CSS, JS, and HTML is in a single `index.html` file. No dependencies except Google Fonts (DM Sans + DM Serif Display).
